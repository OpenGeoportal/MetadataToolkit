Citation and responsible party information (CIT) Version: 1.0

Description

CIT 1.0 is an XML Schema implementation derived from ISO ISO 19115-1, Geographic Information - Metadata - Part 1: Fundamentals, Clause 6.6.3. It includes utility elements for citation, identification, and web linkage of resources. The XML schema was encoded using the rules described in ISO/TS 19139:2007, Clause 8 and ISO/TS19115-3, Clause 8.

XML Namespace for cit 1.0

The namespace URI for cit 1.0 is http://www.isotc211.org/namespace/cit/1.0.

XML Schema for cit 1.0

cit.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the cit 1.0 namespace or by XML Schema documents importing the cit 1.0 namespace. This XML schema includes (indirectly) all the implemented concepts of the cit namespace, but it does not contain the declaration of any types.

NOTE: The XML Schema for cit 1.0 are available here as part of a zip archive including all the XML Schema Implementations defined in ISO/TS 19115-3.

Related XML Schema for cit 1.0

citation.xsd implements the UML conceptual schema defined in ISO 19115-1, 6.6.3. It was created using the encoding rules defined in ISO 19118, ISO 19139, and ISO 19115-3 and contains the following classes: CI_Address, CI_Citation, CI_Contact, CI_Date, CI_DateTypeCode, CI_Individual, CI_OnLineFunctionCode, CI_OnlineResource, CI_Organisation, AbstractCI_Party, CI_PresentationFormCode, CI_Responsibility, CI_RoleCode, CI_Series, CI_Telephone, and CI_TelephoneTypeCode

No Related XML Namespaces for cit 1.0
